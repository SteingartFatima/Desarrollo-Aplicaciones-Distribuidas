Protocolos de Red

  1. HTTP: Es el protocolo usado para cargar páginas web y recursos desde el servidor al navegador en texto sin cifrar.
  2. HTTPS: Es la versión segura de HTTP, con cifrado TLS/SSL para proteger los datos entre el navegador y el servidor.
  3. FTP: Protocolo para subir y bajar archivos entre un cliente y un servidor (usado en mantenimiento de sitios web y transferencia de grandes ficheros).
  4. DNS: Sistema que traduce nombres amigables (ej. www.ejemplo.com) en direcciones IP para encontrar servidores en internet.
  5. DHCP: Servicio que asigna direcciones IP temporales a los dispositivos de una red automáticamente, sin configurarlo manualmente en cada uno.
  6. TCP: Protocolo de transporte orientado a conexión que garantiza la entrega ordenada y sin errores de los datos entre aplicaciones.
  7. UDP: Protocolo de transporte sin conexión, más rápido que TCP, usado en video, voz y juegos donde la velocidad es más importante que la exactitud total.
  8. IP: Protocolo de red que identifica y enruta paquetes entre dispositivos mediante direcciones IP (es la base de Internet).
