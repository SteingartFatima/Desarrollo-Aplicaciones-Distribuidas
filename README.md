## 🎯 Objetivo general

Desarrollar habilidades de maquetación web con HTML y CSS, construyendo interfaces relacionadas con el ámbito de las redes de datos y las telecomunicaciones.

---

## 📌 Objetivos específicos

Al finalizar este trabajo práctico, el estudiante será capaz de:

- Estructurar contenido web semántico con HTML.
- Aplicar estilos básicos e intermedios con CSS.
- Organizar información técnica de forma clara y visual.
- Diseñar páginas simples vinculadas a redes, dispositivos, protocolos, topologías y servicios de telecomunicaciones.
- Utilizar buenas prácticas de presentación y organización visual.
